A Machine Learning based web application to predict diabetes risk using patient health parameters.

# Features
- Takes input: Age, Gender, Glucose, BMI, Blood Pressure, Insulin, Pregnancies, etc.
- Predicts: Non-Diabetic / Pre-Diabetic / Diabetic
- Interactive Streamlit UI with risk visualization

# Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit
- Plotly
